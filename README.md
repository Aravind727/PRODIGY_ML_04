# PRODIGY_ML_04

## Hand Gesture Recognition using Machine Learning

This project implements a Hand Gesture Recognition system capable of identifying and classifying different hand gestures from image data using Machine Learning and Computer Vision techniques.

## Objective

Develop a model that can accurately recognize and classify different hand gestures from image data.

## Dataset

LeapGestRecog Dataset

https://www.kaggle.com/datasets/gti-upm/leapgestrecog

## Technologies Used

- Python
- OpenCV
- NumPy
- Scikit-Learn
- Matplotlib

## Features

- Image preprocessing
- Gesture classification
- SVM model training
- Performance evaluation
- Prediction visualization

## Files

- hand_gesture_recognition.py
- check_dataset.py
- requirements.txt
- gesture_predictions.png
- README.md

## Outcome

Successfully trained a machine learning model to classify different hand gestures from image data.
